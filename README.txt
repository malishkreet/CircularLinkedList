Небольшие уточнения.
--------------------------------------------------------------------------------
runTests.sh - скрипт, который компилирует и запускает юнит тесты.
Пример использования:
./runTests.sh
(Возможно, для запуска необходимо будет сначала исполнить chmod +x runTests.sh)
--------------------------------------------------------------------------------
compileProgram.sh - скрипт, который компилирует итоговую программу, запустить ее надо потом самостоятельно, передав нужный аргумент (имя файла).
Пример использования (и компиляция и запуск итоговой программы):
./compileProgram.sh
./program
(Возможно, для запуска необходимо будет сначала исполнить chmod +x compileProgram.sh)
--------------------------------------------------------------------------------
В main.cpp содержится код, который обрабатывает ввод пользователя, и работает с циклическим списком;
В CircularLinkedList.h, CircularLinkedList.cpp содержатся объявлени и опредление класса CircularLinkedList;
В CircularLinkedListTests.h, CircularLinkedList.cpp содержатся юнит-тесты, необходимые для проверки корректности работы класса CircularLinkedList.
!!! И main.cpp и CircularLinkedListTests.cpp содержат функцию main, файлы предназначены для сборки двух разных исполняемых файлов
(один для программы работы со списком, другой для запуска тестов).
